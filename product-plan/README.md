# MbareteGainz — Export Package

This package contains everything you need to implement MbareteGainz, a fitness tracking app for logging workouts, meals, and weight.

## Quick Start

### Option 1: One-Shot Implementation

Copy the contents of `prompts/one-shot-prompt.md` to your coding agent along with `instructions/one-shot-instructions.md` and `product-overview.md`.

### Option 2: Incremental Implementation

Use `prompts/section-prompt.md` as a template, providing one milestone at a time from `instructions/incremental/`.

## Implementation Order

1. **Foundation** — Project setup, Tailwind, fonts
2. **Daily Check-In** — Core logging functionality
3. **Progress Dashboard** — Visualization and charts
4. **Workout Tracking** — Strength progress tracking
5. **Correlation Insights** — Pattern analysis

## Tech Stack (Recommended)

- **Framework**: Next.js 14+ or Vite + React
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **State**: React hooks (useState, useEffect)
- **Types**: TypeScript

## Design System

| Role | Color | Font |
|------|-------|------|
| Primary | `lime` | DM Serif Display (headings) |
| Secondary | `amber` | Inter (body) |
| Neutral | `zinc` | JetBrains Mono (numbers) |

## Components

All components are:
- **Props-based**: Data and callbacks passed via props
- **Portable**: No external dependencies beyond Lucide icons
- **Responsive**: Mobile-first with Tailwind breakpoints
- **Dark mode ready**: Uses `dark:` variants
