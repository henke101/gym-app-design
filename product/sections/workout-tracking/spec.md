# Workout Tracking Specification

## Overview
A progress tracking dashboard for strength exercises, showing how weight lifted and training volume have improved over time. Users see all their exercises at a glance and can filter by custom date ranges to analyze their progression.

## User Flows
- View overview dashboard showing all strength exercises with progress indicators
- Select custom date range to analyze a specific training period
- See weight lifted over time chart for each exercise
- See volume (sets × reps × weight) trends for each exercise

## UI Requirements
- Overview dashboard layout with all strength exercises visible
- Mini progress charts or indicators for each exercise on the overview
- Custom date range picker for filtering data
- Charts showing weight progression over time
- Charts showing volume trends over time
- Strength exercises only: Chest, Lower Back, Upper Back, Shoulders, Arms, Legs, Core (no Cardio/Walking)
